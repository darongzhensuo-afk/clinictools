# 台灣健保 SGLT2i 給付規範臨床摘要 (2026 更新)

本摘要整合台灣健保署 (NHI) 對 SGLT2 抑制劑 (如 Forxiga, Jardiance) 的最新給付規定。自 **2025 年 3 月 1 日** 起已全面擴大給付至非糖尿病之慢性腎臟病與心衰竭患者。

## 一、 第二型糖尿病 (T2DM) 給付標準
需符合以下所有條件：
1.  **對象：** 第二型糖尿病患者。
2.  **用藥順序：** 需先使用過最大耐受劑量的 **Metformin** 仍無法理想控制血糖者。
3.  **腎功能限制：** eGFR 需 **≥ 30** mL/min/1.73m² (依仿單建議)。
4.  **併用限制：** 
    *   **不可**與 GLP-1 RA (如 Ozempic, Trulicity) 併用。
    *   口服降血糖藥物成分總計以最多 **4 種** 為限。

## 二、 心臟衰竭 (Heart Failure) 給付標準
不論是否有糖尿病，符合以下條件可健保給付：
1.  **左心室射出分率：** 
    *   **LVEF ≤ 49%** (包含 HFrEF 與 HFmrEF)。
2.  **臨床症狀：** NYHA Class II 至 IV 級。
3.  **基礎治療：** 需已接受標準治療 (ACEI/ARB/ARNI 及 Beta-blocker) 穩定劑量至少 **4 週**，仍有症狀者。
4.  **適用藥品：** Dapagliflozin 10mg 或 Empagliflozin 10mg。

## 三、 慢性腎臟病 (CKD) 給付標準
不論是否有糖尿病，符合以下條件可健保給付：
1.  **腎功能指標：** **eGFR 25 至 60** mL/min/1.73m²。
2.  **尿蛋白指標：** **UACR 200 至 5000** mg/g。
3.  **行政條件：** 需加入 **Early-CKD** 或 **Pre-ESRD** 照護計畫。
4.  **合併糖尿病者：** 若不符合上述 CKD 專屬指標，仍可依「糖尿病」標準給付。

---
## 臨床判定流程 (SGLT2i Selection)
1.  **Step 1: 檢查 eGFR。** 若 < 25，通常不建議啟動。
2.  **Step 2: 檢查 LVEF。** 若 ≤ 49% 且有症狀，可優先考慮心衰竭給付。
3.  **Step 3: 檢查 UACR。** 若 200-5000 且 eGFR 25-60，確認是否加入 CKD 計畫。
4.  **Step 4: 檢查 DM。** 若以上皆不符但有糖尿病，確認 Metformin 使用情形。
