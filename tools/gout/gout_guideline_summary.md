# 2024 台灣高尿酸血症與痛風治療指引臨床摘要

本摘要根據 2024 年更新版《Taiwanese Association for the Study of Gout and Hyperuricemia (TASGH)》指引整理，重點在於降尿酸藥物（ULT）的啟動時機與治療目標。

## 一、 降尿酸藥物 (ULT) 啟動標準

### 1. 已確診痛風患者 (Gouty Arthritis)
| 臨床情況 | 建議強度 | 啟動標準 (sUA) |
| :--- | :--- | :--- |
| 臨床可見或影像檢出**痛風石 (Tophi)** | 強烈建議 | 不論數值，直接啟動 |
| 影像學顯示**關節破壞 (Radiographic damage)** | 強烈建議 | 不論數值，直接啟動 |
| **頻繁發作** (每年 $\ge 2$ 次) | 強烈建議 | 不論數值，直接啟動 |
| 合併**尿路結石 (Urolithiasis)** | 建議啟動 | 不論數值，直接啟動 |
| **首次發作**且血尿酸極高 | 建議啟動 | sUA $> 9.0$ mg/dL |
| **首次發作**且合併 **CKD Stage 3+** | 建議啟動 | sUA $> 7.0$ mg/dL |
| 合併**退化性關節炎 (OA)** | 2024 更新 | 建議啟動 (不論 sUA) |

### 2. 無症狀高尿酸血症 (Asymptomatic Hyperuricemia)
| 合併共病症 | 啟動標準 (sUA) | 目的 |
| :--- | :--- | :--- |
| 無特定共病 | $\ge 9.0$ mg/dL | 預防首次痛風發作 |
| 合併 **CKD Stage 3+** | $> 7.0$ mg/dL | **延緩腎功能惡化** (重要更新) |
| 合併 **HTN, DM, Metabolic Syndrome** | $\ge 8.0$ mg/dL | 降低心血管與代謝風險 |
| 合併 **Ischemic Heart Disease** | $\ge 8.0$ mg/dL | 降低心血管風險 |

---

## 二、 治療目標 (Treat-to-Target)

*   **一般目標值：** sUA **$< 6.0$ mg/dL**。
*   **有痛風石者：** 建議初期控制在 **$< 5.0$ mg/dL**，以加速痛風石溶解。
*   **長期維持：** 建議長期維持在目標值以下，以減少發作並預防共病。

---

## 三、 降尿酸藥物選擇建議 (Medication Selection Logic)

在決定啟動 ULT 後，應根據患者的共病症與器官功能進行選擇：

### 1. 藥物特性與首選邏輯
| 藥物名稱 | 作用機制 | 建議首選情境 | 禁忌與風險 |
| :--- | :--- | :--- | :--- |
| **Febuxostat** | XOI (黃嘌呤氧化酶抑制劑) | **CKD Stage 3+**、肝硬化、或對傳統藥物有禁忌者。 | 有重大心血管疾病史者需謹慎評估。 |
| **Allopurinol** | XOI (黃嘌呤氧化酶抑制劑) | 傳統首選，價格低廉。 | **HLA-B\*5801** 過敏風險 (SJS/TEN)。建議檢測基因，CKD 患者劑量需減量。 |
| **Benzbromarone**| Uricosuric (促尿酸排泄) | 腎功能尚可且無結石者。 | **禁忌：尿路結石 (Urolithiasis)**、重度腎功能不全 (eGFR < 30)。 |

### 2. 台灣健保給付規範 (2021)
*   **第一線給付 Febuxostat 之條件：**
    1. 慢性腎臟病 (CKD) Stage 3 以上。
    2. 肝硬化。
    3. Allopurinol 或 Benzbromarone 使用無效或有禁忌。
*   **一般給付：** Allopurinol 或 Benzbromarone 具第一線給付資格。
